# Keyboard Shortcuts Reference

## Global Navigation Shortcuts

These shortcuts work from anywhere in the application:

| Shortcut | Action |
|----------|--------|
| `Ctrl+Alt+H` | Go to Home page |
| `Ctrl+Alt+G` | Go to Generate page |
| `Ctrl+Alt+E` | Go to Edit page |
| `Ctrl+Alt+L` | Go to Library page |
| `Ctrl+Alt+M` | Go to Marketplace page |

## Editor Shortcuts

These shortcuts work when you're in the Package Editor:

| Shortcut | Action |
|----------|--------|
| `Ctrl+N` | New Package |
| `Ctrl+O` | Open Package |
| `Ctrl+S` | Save Package (when changes exist) |
| `Ctrl+E` | Export Package (when package is loaded) |

## Tips

- Global navigation shortcuts use `Ctrl+Alt` to avoid conflicts with browser/system shortcuts
- Editor shortcuts use `Ctrl` alone for common operations
- Shortcuts are case-insensitive
- Disabled shortcuts (e.g., Save when no changes) will not trigger

## Future Enhancements

Planned keyboard shortcuts for future releases:

- `Ctrl+Z` / `Ctrl+Y` - Undo/Redo
- `Ctrl+F` - Search/Find
- `F5` - Refresh/Reload
- `Escape` - Close dialog/Cancel operation
- `Ctrl+K` - Command palette

---

**Note:** On macOS, `Cmd` key is used instead of `Ctrl` for most shortcuts.
