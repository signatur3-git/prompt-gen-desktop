## Analysis Patterns (REVISED)

## Analysis Patterns (REVISED)

### Common Elements Across Categories (Expanded)

**Traditional Elements (Categories 1-7):**
- Subjects: Characters, creatures, objects, natural features, structures
- Modifiers: Age, size, condition, quality, color
- Context: Time, weather, lighting, location, atmosphere
- Composition: Framing, focus, perspective

**Stylistic Elements (Categories 8-14):**
- Art styles: Low-poly, pixel art, propaganda poster, glitch, comic
- Technical constraints: Limited palettes, resolution limits, dithering
- Medium simulation: Woodblock print, screen printing, CRT display
- Presentation format: Instruction manual, infographic, blueprint
- Perspective tricks: Fish-eye, through objects, unusual angles

### New Datatypes Required (Beyond Traditional)

**Art Style Modifiers:**
- Rendering styles: low-poly, pixel art, vector, voxel, flat shaded
- Color constraints: 4-color palette, 16-bit, monochrome, duo-tone
- Technical effects: scanlines, dithering, compression artifacts
- Historical styles: constructivist, art nouveau, ukiyo-e, bauhaus

**Medium-Specific:**
- Print techniques: woodblock, screen print, lithograph, halftone
- Display types: CRT, LCD, arcade monitor, phosphor green
- Material simulations: blueprint paper, parchment, graph paper, newsprint

**Presentation Formats:**
- Instructional: assembly diagram, field guide, technical manual
- Data visualization: infographic, flowchart, map legend
- Sequential: comic panel, storyboard, step-by-step
- Surveillance/Interface: security cam, HUD overlay, terminal screen

**Perspective Techniques:**
- Unusual angles: ant's eye, straight down, through objects
- Optical effects: fish-eye, kaleidoscope, reflection in surface
- Frame devices: keyhole, window, scope, mirror

## Magic Sauce Identification

### What Makes These Categories Special

**Game Art Styles:**
- Nostalgia factor (people love retro)
- Technical constraints create unique aesthetic
- Gamers are huge AI art audience
- Different from typical "photorealistic fantasy"

**Technical/Instructional:**
- Rarely seen in AI art generators
- Practical and fun
- Shows system can do structured content
- Educational value

**Propaganda/Period:**
- Strong visual identity
- Historical interest
- Political/social commentary potential
- Distinct from generic fantasy
